# Task 10: Vue Components & Bootstrap Integration

**Student:** Omar Antonio Samano Perez

## Overview

This project builds upon the previous work by refactoring the Chuck Norris jokes application into a component-based architecture using Vue.js and styling it with the Bootstrap 5 framework for a fully responsive user experience.

## Features Implemented

- **Vue.js Components:** Developed a reusable `<chuck-card>` component to encapsulate the joke display logic, improving code modularity and maintainability.
- **Data Communication (Props):** Implemented property passing (props) to successfully transfer `icon_url` and `value` data from the parent Vue instance to the child components.
- **Responsive Grid System:** Leveraged Bootstrap's grid classes (`row`, `col-12`, `col-md-4`) to ensure a dynamic layout that displays 3 cards per row on desktop and 1 card per row on mobile devices.
- **Bootstrap UI:** Utilized the Bootstrap Card component with images and text to present the data in a clean, professional, and consistent visual format.

## Technologies & Resources Used

- **Backend Framework:** Laravel 12
- **Frontend Library:** Vue.js (via CDN)
- **CSS Framework:** Bootstrap 5.3.0
- **Template Engine:** Laravel Blade
