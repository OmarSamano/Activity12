# Activity 12: ChuckJokesVue

**Student:** Omar Antonio Samano Perez

## Overview

This project demonstrates the integration of the Vue.js JavaScript framework within a modern Laravel 12 application. It utilizes a primary layout with Blade template inheritance to render a dynamic, stylized, and responsive list of data.

## Features Implemented

- **Vue.js Integration:** Created a new Vue instance to manage and return the suggested static data (Chuck Norris jokes).
- **Dynamic Data Rendering:** Displayed the data as a list by iterating through the array using the Vue `v-for` directive.
- **Master Layout & Inheritance:** Built a primary view layout and utilized Blade's `@extends` and `@section` directives to manage the application's structure.
- **Aesthetic UI:** Added custom CSS styling to provide a clean, modern, and minimalist look to the displayed data.

## Technologies & Resources Used

- **Backend Framework:** Laravel 12
- **Frontend Framework:** Vue.js (implemented via CDN for a no-build-step approach)
- **Template Engine:** Laravel Blade
